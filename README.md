NUCLEO-F072RB_eeprom_i2c (PORTUGUÊS)
==============
- Projeto    : NUCLEO-F072RB_eeprom_i2c
- MCU        : STM32F072RB
- Copyright  : Atribuição (CC BY 4.0) Diego Moreno
- Descrição  : Grava uma página de 16 bytes na memória EEPROM M24C08-W via I2C.
